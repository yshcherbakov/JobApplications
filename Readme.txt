Job Applications web application - Yuriy Shcherbakov test project, 09/27/2016

Technologies used:
VS2013, C#, MVC, .NET, Entity Framevork, MS SQL, jQuery, MSTest.

http://thewsoftware.dynu.com/JobApplications.Web.App/

1. Open the web app url in a browser.
2. Register a new applicant user.
3. Complete a job application (unacceptable answers are in red color).
4. Submit application.
5. Login as employer (U:employer, P:employer).
6. View acceptable applications.
